# 🏡 Estate Financial Transactions Tracker  

## 📌 Project Overview  
This **Excel-based financial tracker** is designed to **monitor estate bill payments**, ensuring accurate record-keeping of **electricity, service charges, water bills, and other revenue sources**. It includes automated analysis of key financial metrics to improve financial oversight and transparency.  

## 📊 Features  
✅ **Automated Payment Tracking** – Monitors payments for different estate services.  
✅ **Resident Payment Status** – Identifies **paid, pending, and overdue bills**.  
✅ **Dynamic Dashboard with KPIs** – Provides **insightful analytics** on estate finances.  
✅ **Customizable Reports** – Generates **monthly and yearly revenue summaries**.  
✅ **Data Validation & Accuracy** – Prevents errors in payment entries.  

## 📈 Key Performance Indicators (KPIs)  
📌 **Total Revenue Collected** – Tracks **total payments received** per period.  
📌 **Top-Paying Residents** – Identifies residents contributing the highest revenue.  
📌 **Revenue by Bill Type** – Breaks down payments for **electricity, water, service charge, and others**.  
📌 **Monthly Revenue Trends** – Analyzes revenue flow over time.  

## 📂 Files in the Repository  
- `Estate_Financial_Transactions.xlsx` – The main Excel file containing payment records, KPIs, and the dashboard.  
- `README.md` – Documentation explaining project details.  

## 🛠 How to Use  
1. **Download the Excel file** from this repository.  
2. Open in **Microsoft Excel (2016 or later)**.  
3. Enter **payments received** for different estate services.  
4. View **automated calculations & KPIs** in the dashboard.  
5. Analyze **payment trends** and track **overdue payments**.
